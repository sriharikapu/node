# mnist-utility

## 1. Installation
1. [IPFS](https://docs.ipfs.io/guides/guides/install/)
2. Python Dependencies ```pip install -r requirements.txt```

## 2. Setup
```
ipfs init
```

## 3. Running the Code
```
ipfs daemon
```

## 4. Code Linting
```
pylint --rcfile=pylintrc ../mnist_utility/
```

## 5. Update Submodules
```
git submodule update --recursive
```
